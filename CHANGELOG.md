# Change Log

## 0.5.0

### Minor Changes

- [`263310f`](https://github.com/cbnsndwch/remix-v3-html-highlighter/commit/263310fb148a0f0745bbdafe38b18020e8237fa9) Thanks [@cbnsndwch](https://github.com/cbnsndwch)! - - Added support for bare `html` tagged templates.
